Данное приложение _сосотоит из 2 файлов Server_gui.exe и Client_gui.exe которые запускаются на разных компьютерах, для передачи информации используется безпроводная 
Wifi сеть. Оба компьютера должны быть подключены к одной сети.
Запуск
1)Запускается приложение Server_gui.exe Выбираются роли для участников и реализуемый протокол и нажимаем Submit. 
Приложение извлекает ip адресс безпроводного интерфейса и ждет подключения. Ввывод на экран (Старт сервера на ip <ip> порт 10000)

2)Запускаем приложение Client_gui.exe, вводим ip адресс второго компьютера для подключения.
3)После окончания работы одного протокола можно выбрать следующий или закрыть приложение.

Список реализуемых протоколов.
1)Привязка к биту на основе симметричной криптографии
2)Протокол подбрасывания монеты на основе однонаправленной функции
3)Экспоненциальный протокол подбрасывнания монеты

Ознакомится со всеми протоколами можно по ссылке http://www.cryptoprotocols.kz/index.php?view=examples
